# Z-Env
Z-Env (aka zenv) is a simple environment manager for your shell.

The main goal of this project is to provide a simple way to manage your environment variables, 
making it easier to switch between different development environments and configurations.

### The main features of Z-Env are

- Detect directory changes and automatically reload environment variables
- List of allowed paths to load environment variables from
- Support for multiple environment files: it can (re)load all `.env` files from the current directory and its parents
- Support for multiple shells: zsh and bash (others can be added if you make the necessary hooks) 

This project is still in development and is not yet ready for use.

Contributions are welcome!
